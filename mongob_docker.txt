Run MongoDB Community Server latest version locally:

(1) Install docker (engine or desktop).
        https://docs.docker.com/engine/install/ubuntu/

(2) Download the official MongoDB docker image
        sudo docker pull mongodb/mongodb-community-server:latest

(3) Create a local directory for the image to persist the data, with write permissions for all users
        sudo mkdir -p /mongodb/bluezone/data
        sudo chmod -R 777 /mongodb

(4) Create and run a docker container with the downloaded image
        sudo docker run --name mongodb_container -d -p 27017:27017 -v /mongodb/bluezone/data:/data/db mongodb/mongodb-community-server:latest

(5) Connect to the MongoDB server:
        sudo docker exec -it mongodb_container mongosh

(6) Create the bluezone db running these commands:
        use bluezone
        db.createCollection("tickets")
        db.createCollection("sequences")
        db.createCollection("rates")
        db.sequences.insertOne({"name":"ticket-code","nextValue":Long("1")})
        db.rates.insertMany([{"name":"Blue","eurosPerHour":"0.80"},{"name":"Orange","eurosPerHour":"0.85"}])
